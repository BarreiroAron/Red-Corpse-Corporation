o - Descripcion del videojuego: Un juego online en primera persona. Estas encerrado en un lugar en total oscuridad y aparentemente irreconocible.
                                      Notas leves luces tenues que salen del techo y alguna que otra vidriera por encima que parecen ser ventanas,
                                  frente a ti, encuentras una persona la cual solo se le ve el rostro, un locutor te va a hablar sobre un juego de cartas.
                                      Ahora, estas destinado a jugar y ver que te depara el futuro en esta mesa de casino.

o - IDE: La IDE que vamos a estar usando para estre proyecto va a ser IntelliJ
o - Tecnologías Principales: - LibGDX va a ser la libreria que usemos para programar con java nuestro videojuego.
                            - Nuestro videojuego va a estar hecho para ser jugado en computadoras de escritorio/notebooks en la web mediante internet.

o - Enlace a la Wiki del Proyecto: [Red Corpse Wiki](https://github.com/BarreiroAron/Red-Corpse-Corporation/wiki)

o - Instrucciones Básicas para importar el proyecto: 
- Paso 1: Descargar el proyecto en formato .rar
- Paso 2: Extraer el proyecto .rar
- Paso 3: abrir un IDE (Eclipse por ejemplo)
- Paso 4: Una vez hecho esto, se va a tener que ir a "File" (se encuentra arriba a la izquierda),
- Paso 5: La opcion llamada "Import..."
- Paso 6: La carpeta llamada "Gradle"
- Paso 7: y le das a "Exising Gradle Project". Luego de eso, te deberia hacer esperar un rato para la imoprtacion y ya podrias ejecutarlo tranquilamente
- Paso 8: busca el módulo/subproyecto de escritorio. Nota: Con Liftoff, en general se llama lwjgl3 en lugar de desktop. 
- Paso 9: Dentro de ese módulo, busca la clase lanzadora (ej. Lwjgl3Launcher.java) y ejecutala (Run As > Java Application o similar).
- Paso 10: Si se abre una pantalla con una imagen de LibGDX entonces esta perfecto, sino, Es posible que necesites configurar el "Working Directory" en la configuración de ejecución para que apunte a la carpeta assets/ del proyecto core/ si las imágenes no cargan. 

  - Ya se deberia tener el juego para ejecutarlo en tu IDE
